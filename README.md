# Forecasting Crop Yields Using Statistical Methods

## Overview

This project is a group academic study focused on forecasting agricultural crop yields using multi-factor environmental and agronomic data. The objective is to model yield variations using statistical and machine learning techniques and identify optimal input conditions for improved agricultural productivity.

The study combines exploratory data analysis, regression modeling, time-series analysis, and Bayesian optimization to understand complex relationships between agricultural inputs and crop yield.

---

## Project Year: 2025

## Objectives

* Analyze the relationship between crop yield and environmental/agronomic factors
* Perform exploratory data analysis on agricultural datasets
* Develop predictive models for yield forecasting
* Identify optimal input conditions for maximizing yield
* Apply statistical and machine learning methods for decision support

---

## Data

The dataset includes multi-factor agricultural variables such as:

* Rainfall
* Temperature
* Fertilizer usage
* Pest incidence
* Technology adoption index

These variables are used to model and forecast crop yield variations.
